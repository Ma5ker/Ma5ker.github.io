Kali linux下安装网易云音乐

发表于 2017-04-26 更新于 2020-01-09 分类于 Just for fun
本文字数： 587 阅读时长 ≈ 1 分钟

  网易云音乐出了Linux下版本，由于电脑装的Kali Linux版本，而网易云官网所提供的并没有debian版本的网易云音乐安装包，安装时出现环境依赖无法解决。找了好久找到一篇通过修改deb包成功在kali上安装网易云的大佬的博客，记录如下备份。

安装包下载

下载地址：netease-cloud-music
安装包根据netease-cloud-music_1.0.0-1_amd64_deepin15.deb修改

安装

dpkg -i netease-cloud-music_1.0.0-1_amd64.kali.deb
apt-get install -f
dpkg -i netease-cloud-music_1.0.0-1_amd64.kali.deb

修改启动命令

装过chrome的应该都知道，有些应用比如chrome并不支持直接在系统中以root身份启动，会出现报错信息，由于kali linux默认root身份登录，故如同chrome，应在启动命令中添加–no-sandbox参数
vim /usr/share/applications/netease-cloud-music.desktop

修改：

Exec=netease-cloud-music %U

为

Exec=netease-cloud-music %U --no-sandbox

之后便可以双击图标运行了。
至此安装完成。
